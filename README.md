# DVORAK Layout Setup for Keychron J4.

### It is assumed that your keycaps are physically arranged in DVORAK.


Configuring a DVORAK layout on your keyboard can often be complicated. Operating systems typically assume QWERTY by default, and switching layouts can be cumbersome—sometimes requiring a second keyboard to complete the setup.

## Firmware Configuration
You can apply this firmware configuration through: [launcher.keychron (firmaware app)](https://launcher.keychron.com/)

## Layers
- **Layer 0** → Use when your operating system is configured as **DVORAK**
- **Layer 2** → Use when your operating system is configured as **QWERTY**

## Switching Layers

To switch between layers, press:
 - FN + ESC
